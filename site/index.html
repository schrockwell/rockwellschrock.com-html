About this site

As a full-stack developer, I spend my days fighting with complicated markup and fragile build tools. So why should I spend my free time doing that, too?

This site is build with YASS, a tiny static site generator written in Go. It has no JS, a pinch of CSS, and it's served up via HTTP, so it's usable on older browsers – even the really old ones.
